h1. Elasticsearch WebHooks Plugin

p. This plugin attaches WebHooks to various asynchronous HTTP operations.

h2. Events

*Event:* created
*Description:* p. Fires when an index is opened or created. Returns the name of the index created.

*Event:* closed
*Description:* p. Fires when an index is closed. Returns the name of the index closed.

h2. Configuration

code. webhooks:
    callback: http://localhost:1111
